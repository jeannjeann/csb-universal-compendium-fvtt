## インストール

FVTTの「モッド・拡張機能」の「モジュールを入手」ウィンドウで、「URLを指定」の欄に「ManifestURL」をペーストしてインストールしてください。

必須モジュールのインストールが促されるので、一緒に導入してください。

- ManifestURL: https://github.com/jeannjeann/csb-universal-compendium-fvtt/releases/latest/download/module.json
