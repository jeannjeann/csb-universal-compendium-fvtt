# 使い方

FVTTの基本的な使い方は、[公式ページ](https://foundryvtt.com/)、[日本語Wiki](https://foundryvtt.wiki/ja/home)、[Discordサーバ](https://discord.com/invite/vM4YM27)、他インターネット上にさまざまな情報があります。参考にしてください。


## ワールド作成

　FVTTの使い方を参考に作成してください。モジュールなどで以下の要素に関して補助しています。
- 「Share Media」モジュールを使うことで、シーンの一部に表示指定ある画像を簡単に変更できます。
- 「Cut-in Manager」モジュールを使うことで、さまざまなトリガーに反応するカットインを設定できます。
- 「Simplefog - Manual Fog of War」モジュールを使うことで、マップに自由にマスクをかけられます。
- 「Monk's Active Tile Triggers」モジュールを使うことで、マップ上のタイルにさまざまなトリガーを作成できます。

#### 複数のワールドで使用できる辞典
- FVTT初期画面の「モッド・拡張機能」タブにある「モジュールを作成」ボタンから、複数のワールドで共通して使える辞典を用意できます。データが増えていけばいろいろと使い回せるようになるので、ワールド作成が楽になっていきます。


## BCDiceコマンド

　「BCDice for Foundry VTT」モジュールの機能です。詳細はモジュールの公式ページを参照してください。

#### BCDiceダイアログからの使用
- シーンコントロールボタン（画面左）、アクターシートまたはトークンシートのタイトルバーにある「BCDice」で表示されるダイアログから実行できます。

#### チャット入力欄からの使用
- 「/bcd」の後に半角スペースを空け、BCDiceコマンドを入力することで実行できます。

### チャットパレット
- インポートまたは手動作成が可能です。

#### 変数の使用
- BCDiceでのロールに使う変数にはキャラクターシートのKeyを指定してください。例:1d20+{s0}+{p0}
- キャラシサイトからインポートした場合などですでに変数が組み込まれている場合、その変数とKeyを結びつける記述をBCDiceの変数設定に追加することで、インポートしたコマンドの変数を変更せずにそのまま使うこともできます。例：筋力={p0}
- BCDiceの変数設定に「**リンク**」の記載は同期するために必須です。インポートしたり改変したりすることで消えてしまった場合は必ず追記してください。

#### Token Action HUD
- チャットパレットを用意してあれば、「Token Action HUD」モジュールを使用することでHUDからマウス操作でコマンドを実行できます。
- さまざまにカスタマイズが可能です。詳細はモジュールの公式ページを参照してください。


## キャラクター

### 作成時に必要な設定
- Templateから「_UW-Character」を選択して「リロード」ボタンで適用する。
- タイトルバーの「初期コマ」ボタンから、関係性を設定する。
- タイトルバーの「初期コマ」ボタンから、リソースタブのバー1を「attributeBar.bar1」、バー2を「attributeBar.bar1」に設定する。
- タイトルバーのBCDiceダイアログの変数に「**リンク**」の内容をコピーする。
- チャットパレットの設定

　これらを設定したひな型となるキャラクターを作成し、そのキャラクターを複製して使用すればこれらの設定を何度も行わなくてよくなります。

### キャラクターシート

#### ヘッダー
- イニシアティブのラベルをクリックすることで、設定したダイスロールが実行されて結果がイニシアティブ欄に反映されます。
- イニシアティブ欄に手動で入力することもできます
- FVTTの戦闘タブでのイニシアティブをロールすると、キャラクターシートのイニシアティブの値が設定されます。

#### ステータスタブ
- ステータス、パラメータは自由に追加できます。
- ステータスには最大値を設定でき、トークンに表示されるバーの候補になります。
- Bar1、Bar2でトークンに表示されるバーの選択してください。
- ステータス、パラメータの数値を変更すれば、BCDiceの変数も変更され、ロールにも反映されます。

#### アイテム・エフェクトタブ
- キャラクターが所持するアイテムとキャラクターに付加されているエフェクトが表示されます。

#### メモタブ
- キャラクターシートサイトのURLを入力できます。
- クリックすることで、キャラクターシートサイトを表示できます。
- その他のメモもできます。


## アイテム
- 数量、説明、エフェクトを設定できます。
- アイテムに設定したエフェクトは、そのアイテムを所持したキャラクターに自動的に付加されます。

## エフェクト（Active Effect）
- ワールドの設定タブの「Configure Active Effect」からエフェクトを管理できます。
- バフの有無のメモなどに使用することを想定しています。
- CSBの仕様上、ステータスとパラメータの数字を操作することはできません。
- CSBテンプレートの改変とチャットパレットの工夫で数字操作を可能にすることはできます。

## ステータス効果（Status Effect）
- 状態異常はトークン右クリックの「ステータス効果の設定」で表示される効果です。
- ワールドの設定タブの「Configure Active Effect」からステータス効果を追加、編集、削除できます。
- 各システムにあわせて編集することで、便利に使えます。
