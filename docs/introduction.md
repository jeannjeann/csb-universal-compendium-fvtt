# Custom System Builder 汎用ワールド

　これは、「Custom System Builder」と「BCDice for Foundry VTT」を利用して、汎用的に使えるワールドセットです。

　国内よく使われているオンセツールと同程度の機能と使い勝手を実装し、FVTTらしさも多少盛り込んだワールドになっています。BCDiceが対応しているシステムや普通のダイスロールしか使わないシステムを手軽にFVTTで遊べるようにすることを目的としています。このワールドをベースにCSBテンプレートを改変したり、モジュール環境を変更したりして、オリジナルのワールドを作ることも可能です。

　同等の機能の実装、使い勝手の向上のために、推奨モジュールを用意しています。



## 使い始めるまでの流れ

### 1.初期設定

**リンク**

　汎用ワールドが機能するように、初期設定を行います。このワールドではほとんどがすでに設定済ですが、一度確認しておくことをオススメします。

### 2.ワールドの複製とシステムにあわせた環境構築

**リンク**

　実際にセッションで使用するワールドを、初期設定が完了しているワールドから複製することを推奨します。

　使用するシステムにあわせてモジュールの取捨選択や設定などを行い、セッションのベースとなるワールドを作成します。汎用的な演出などもここで設定しておくと後が楽です。基本的なモジュールは同梱していますが、FVTTにはリッチな演出を助けるモジュールや使い勝手を向上させるモジュールがたくさんあるので、いろいろ試してみましょう。各モジュールの使い方は、それぞれのモジュールの公式ページを参照してください。

　システム毎にセッションベースとなるワールドを保存し、実際のセッション用のワールドは複製して作成すると使い回せるので後が楽になります。

### 3.各シナリオ用のワールド作成

**リンク**

　シナリオの内容を追加していき、セッションにあわせてワールドを整えてください。使い方を参考に、キャラクターやアイテムを作成してください。シナリオ特有の演出などもここで設定しておきましょう。

### 4.キャラクターなどの作成とセッション開始

**リンク**

　プレイヤーキャラクターを作成し、ダイスロールなどの操作方法を理解したらセッション開始です。
