# 設定

## Custom System Builderの設定（設定済）
- Initiative Formulaを「```[${initiative}$]```」にする。

## BCDice for Foundry VTTの設定（設定済）
- CustomSystemBuilder連携を有効にする。
- 同期対象変数設定に「**リンク**」の内容をコピーする。

## キャラクターテンプレートの設定（システム毎に変更必要）
- 「_UW-Character」のイニシアチブをクリックしてダイアログを開く。
- 「Roll message」の欄にある「1d20 イニシアティブ」を各システムにあわせたロール式に変更する。直接入力する場合や使用しない場合は空欄でもかまいません。

## キャラクターの設定（アクター作成ごとに設定必要）
- Templateから「_UW-Character」を選択して「リロード」ボタンで適用する。
- タイトルバーの「初期コマ」ボタンから、関係性を設定する。
- タイトルバーの「初期コマ」ボタンから、リソースタブのバー1を「attributeBar.bar1」、バー2を「attributeBar.bar1」に設定する。
- タイトルバーのBCDiceダイアログの変数に「**リンク**」の内容をコピーする。

## アイテムの設定（アイテム作成ごとに設定必要）
- Templateから「_UW-Item」を選択して「リロード」ボタンで適用する。

## コピー元になるキャラクターとアイテムの作成のススメ

　キャラクターは基本的なステータスとパラメータを設定し、上記の設定を済ませたひな型となるキャラクターを作成しておき、それを複製することで新規にキャラクターを作成するようにすれば設定の手間が省けます。アイテムも同様にひな型作成をしておくと便利です。

## その他のコア設定・モジュール設定

　好みに合わせて各種設定を行ってください。

- Monk's Active Tile Triggers：タイルをトリガーにしてイベントを発生させる機能を提供します。
- Multiple Chat Tabs：チャットタブ機能を提供します。
- Pin Cushion：地点メモの使い勝手を改善します。
- Quick App Scale：各ウィンドウの拡縮機能を提供します。

## カスタマイズの指針

### モジュールの追加・変更・削除

　推奨モジュールは自由に変更できます。気に入ったモジュールを追加したり、似たような機能を持つモジュールに入れ替えたり、不要なモジュールを削除したりして使いやすいようにカスタマイズしてください。必須モジュールは動作に必要なものがほとんどなので、なるべく有効にして使うことをオススメします。

### BCDice同期設定の変更

　同期設定はデフォルトではステータス10個、パラメータ20個まで同期するようなっています。必要に応じてBCDiceの同期対象変数設定とアクターのBCDice変数設定を変更して増減してください。

**注意**：変数をカスタマイズした場合、BCDiceコマンドでの変数変更が予想外の挙動になる可能性があります。

### 特定システム用にカスタマイズ

　このワールドやCSBテンプレートを改変することで、特定のシステムに特化させるカスタマイズも可能です。キャラクターテンプレートにさまざまな項目を追加したり、アイテムテンプレートを増やしてスキルや魔法などに対応させたりして使いやすくしてみてください。
